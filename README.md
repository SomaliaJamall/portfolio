# Portfolio
My developement portfolio. Feel free to use as a starting point for your own!

# Stack
Vite
- Easy Developement Server - Builds static HTML/JS/CSS from React code
- - Allows leveraging of React to build the portfolio using flexible industry standard tool while generating code that is SEO friendly

React
- Industry standard front end developement library for JS

# Setup project
1. Install apache on server
2. run `npm install`
3. run `npm run build`
4. Site is served at localhost/index.html
